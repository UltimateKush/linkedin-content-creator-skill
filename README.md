# LinkedIn Content Creator Skill

Claude Code skill for creating LinkedIn posts that don't sound like AI wrote them. Built for GTM engineers.

---

## What it does

This skill helps you write LinkedIn posts and brainstorm content ideas — without the usual AI slop.

- Writes in a conversational, practitioner-first tone
- No corporate jargon, no buzzword soup
- Built specifically for GTM engineers doing cold outbound B2B work
- Covers post types, hooks, CTAs, and style — all baked in
- Designed to help you grow to 10K followers by posting content that actually sounds like you

Whether you have a topic ready or have no idea what to post, the skill walks you through it.

---

## Installation

**1. Clone the repo**

```bash
git clone https://github.com/kushalkhanra/linkedin-content-creator-skill
```

**2. Copy the skill to your Claude skills folder**

```bash
cp -r linkedin-content-creator-skill/linkedin-content-creator ~/.claude/skills/
```

**3. Restart Claude Code**

**4. Type `/linkedin` to trigger it**

---

## How to use it

Open any Claude Code session and type:

```
/linkedin
```

The skill will ask what you want to post about. If you're drawing a blank, it'll ask you a few questions and generate topic ideas for you to pick from.
